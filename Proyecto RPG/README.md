EXpi Online

EXpi Online es un proyecto RPG desarrollado completamente en PSeInt como parte de mi proceso de aprendizaje en programación.

Actualmente el proyecto se encuentra en desarrollo y esta es su primera versión funcional.

---

# Características actuales

- Sistema de inicio de sesión.
- Registro de usuario.
- Selección de clases.
- Menú principal.
- Sistema básico de estadísticas.

---

# Próximas funciones

- Sistema de combate.
- Enemigos.
- Experiencia.

---

# Cómo ejecutar

1. Descarga el proyecto.
2. Instala PSeInt.
3. Abre el archivo `EXpi_V1.psc`.
4. Ejecuta el algoritmo.

---

# Archivo .BAT

El archivo .bat es un acceso rápido para ejecutar el proyecto.

Actualmente solo funcionará si:

PSeInt está instalado en la ruta:
C:\Program Files (x86)\PSeInt\pseint.exe
El archivo .bat se encuentra en la misma carpeta que EXpi_V1.psc.

Nota: El archivo .bat aún está en fase de pruebas y puede no funcionar correctamente en otros equipos.
